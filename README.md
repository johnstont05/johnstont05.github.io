## About

This is my personal portfolio, built to show my work as a data journalist and highlight selected projects with context for why they mattered. It also has a corner for the things I make just for fun.

---

## Built with

- [SvelteKit](https://kit.svelte.dev/)
- [Google Sheets](https://docs.google.com/spreadsheets/) — used as a lightweight CMS for project data, fetched as published CSV
- Deployed on [Netlify](https://netlify.com)
